# CMOS Failure Dashboard

Next.js web dashboard for AI-powered predictive maintenance.

## Setup

```bash
cd dashboard
npm install
npm run dev
```

## Build for Vercel

```bash
npm run build
npm start
```

## Environment

No API keys needed for demo mode. Data is mocked and included.

## Deployment

Push to GitHub, connect to Vercel, set Root Directory to `dashboard/`.

## Structure

- `app/` - Next.js app router (pages)
- `components/` - React components
- `lib/` - Utilities and mock data
- `public/` - Static assets
